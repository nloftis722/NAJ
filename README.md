# NAJ Trucking Tools

A collection of HTML-based calculators and CRM tools for day-to-day use at NAJ Trucking.

## Tools

| Tool | File | Description |
| ---- | ---- | ----------- |
| Example Calculator | calculators/example.html | Short description of what it does |
| Example CRM | crm/example.html | Short description of what it does |

## How to use

These are standalone HTML files. You can open any file by downloading it and opening it in a web browser (Chrome, Edge, etc.) with no installation needed. If GitHub Pages is enabled, each tool is also available at a web link you can bookmark.

## Folder structure

The `calculators/` folder holds calculators such as freight rate, fuel cost, and mileage tools. The `crm/` folder holds customer relationship management tools.
